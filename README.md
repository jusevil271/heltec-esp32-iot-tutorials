# MONTESTEAM - Heltec ESP32 WiFi Kit 32 V3

Repositorio con los ejemplos y tutoriales de la serie dedicada a la **Heltec ESP32 WiFi Kit 32 V3** utilizando **Arduino IDE**.

En este repositorio iré publicando, paso a paso, el código utilizado en los vídeos y en las entradas del blog de MONTESTEAM.

## Contenido

- Pantalla OLED
- Conexión WiFi
- ThingSpeak
- Sensores
- Proyectos IoT
